# Upstage AI Ambassador Project

### 프로젝트 소개
디지털 기기(키오스크) 사용에 어려움을 겪는 어르신 및 노약자들 위한 개인화 맞춤형 AI 비서입니다.
Upstage Solar API와 Document Parse 기술을 활용하여 복잡한 메뉴판 사진을 이해하기 쉬운 텍스트와 음성 대화로 변환해 줍니다.
또한 사용자가 되질문 하기 전에 Solar ai가 자동으로 먼저 추천 메뉴와 가격을 함께 제시하여 사용자의 고민 시간을 줄여줍니다.

### 핵심 기능
1. 인식 : 키오스크 메뉴판 사진을 업로드하면 메뉴명과 가격을 정확히 인식합니다. (Document Parse)
2. 생각 : 인식된 정보를 노약자분들이 보기 편한 표로 정리하고 친절한 말투로 요약해 줍니다. (Solar LLM)
3. 산출 : "가장 싼 게 뭐야?" 같은 질문에 대해 실시간으로 답변합니다.

### 실행 방법
1. API Key 발급 (Upstage Console)
2. `upstage_Task1.py` 코드 내에 API Key 입력
3. 테스트 이미지 업로드 후 코드 실행
